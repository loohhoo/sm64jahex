Japanese SM64 ROM Text ← → Hex Converter Tool

(日本語版はこちら)

I created this tool to both "decrypt" text values (convert the encoding to regular letters) and vice versa. Paste HEX or Japanese on the left textboxes and click either HEX → JA or JA → HEX. Output boxes are readonly. For hard-to-type special characters, use buttons to the right.

Please reference this table for a detailed list of values.

Usage

  • For special characters, paste as you see on this table (minus any explanatory text in brackets []).
  • To add a line break, type lower-case n, and for end dialog, type lower-case e.
  • To add dakuten ('') to a character, insert F0 before it; for handakuten (゜) insert F1 before it.
